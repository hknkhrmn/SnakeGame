## Features

- Classic snake game mechanics
- Score tracking
- High score saved (localStorage)
- Keyboard and touch controls
- Responsive design (mobile friendly)

## Installation

1. Download all files to the same folder:
   - `index.html`
   - `index.js`
   - `style.css`

2. Open `index.html` in your browser

## How to Play

### Keyboard Controls
- **↑** Up
- **↓** Down
- **←** Left
- **→** Right

### Mobile Controls
Use the arrow buttons at the bottom of the screen

### Game Rules
- Move the snake to eat the red food
- Each food increases your score by 1
- Don't hit the walls
- Don't hit your own body
- Try to beat your high score!
